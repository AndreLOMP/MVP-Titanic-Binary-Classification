# MVP-Titanic-Binary-Classification

**ENG**-This project was developed during the Machine Learning & Analytics sprint of the Master’s program in Data Science and Analytics at the Pontifical Catholic University of Rio de Janeiro,  with the aim of analyzing the titanic data containing information about passengers in the ship and create a classification machine to classify if the person on the ship is going to survive or not, based on traits like age, sex, passenger class, etc.

The objective of this study is to carry out the exploratory data analysis, modeling steps and create a classification model for predicting survival on the titanic accident and correlate data with the actual event. 

This project was done with academic intent, so it cant and it WONT be used as reference for real life problems and crisis. 

**PT**-Este projeto foi desenvolvido durante o sprint de Machine Learning e Analytics do Mestrado em Ciência de Dados e Analytics da Pontifícia Universidade Católica do Rio de Janeiro, com o objetivo de analisar os dados do Titanic — contendo informações sobre os passageiros da embarcação — e criar um modelo de classificação para prever a sobrevivência ou não das pessoas a bordo, com base em características como idade, sexo, classe do passageiro, entre outras.

O objetivo deste estudo é realizar a análise exploratória dos dados e as etapas de modelagem, criar um modelo de classificação para prever a sobrevivência no acidente do Titanic e correlacionar os dados com o evento real.

Este projeto foi realizado com fins acadêmicos, portanto, não pode — e não será — utilizado como referência para problemas ou crises do mundo real.
