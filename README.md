# MVP-Titanic-Binary-Classification
This project was developed during the Machine Learning & Analytics sprint of the Master’s program in Data Science and Analytics at the Pontifical Catholic University of Rio de Janeiro,  with the aim of analyzing the titanic data containing information about passengers in the ship and create a classification machine to classify if the person on the ship is going to survive or not, based on traits like age, sex, passenger class, etc.

The objective of this study is to carry out the exploratory data analysis, modeling steps and create a classification model for predicting survival on the titanic accident and correlate data with the actual event. 

This project was done with academic intent, so it cant and it WONT be used as reference for real life problems and crisis.
